# FaculRide

O **FaculRide** é um sistema desenvolvido para facilitar o compartilhamento de caronas entre estudantes universitários, promovendo economia, praticidade e integração entre os alunos.
## 📌 📖 Descrição

O sistema permite que estudantes ofereçam e solicitem caronas dentro do ambiente universitário. A proposta é reduzir custos com transporte, melhorar a mobilidade dos alunos e incentivar a colaboração entre eles.
